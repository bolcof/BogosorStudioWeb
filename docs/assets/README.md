# BogosorStudio Web Assets

Place exported Steam assets here. The site already references these paths:

- `studio/logo.png` - studio logo or wordmark for the header
- `studio/hero.jpg` - slideshow image for the title list slide
- `studio/news.jpg` - slideshow image for the update/news slide
- `studio/og-image.jpg` - social share image, ideally 1200x630
- `games/sak-winik/library_capsule.jpg` - Steam Library Capsule for the Sak Winik shelf card
- `games/sak-winik/library_hero.jpg` - Steam library hero for the first-view feature image
- `games/sak-winik/key_art.jpg` - full-bleed key art for the title detail page
- `games/sak-winik/og-image.jpg` - social share image for the title detail page
- `games/sak-winik/screenshot_01.jpg` - large visual on the title detail page
- `games/sak-winik/screenshot_02.jpg` - secondary visual on the title detail page
- `games/sak-winik/screenshot_03.jpg` - secondary visual on the title detail page
- `games/sak-winik/header.jpg` - optional Steam header image for future sections
- `games/sak-winik/capsule_616x353.jpg` - optional Steam capsule for future sections
- `games/title-02/library_capsule.jpg` - second Steam Library Capsule
- `games/title-03/library_capsule.jpg` - third Steam Library Capsule
- `games/title-04/library_capsule.jpg` - fourth Steam Library Capsule
- `games/title-05/library_capsule.jpg` - fifth Steam Library Capsule

The page works without these files, but it is designed so the real Steam art can be dropped in without changing HTML.
